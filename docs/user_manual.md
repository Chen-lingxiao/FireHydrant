# 校园消防栓管理系统使用手册

## 系统概述

校园消防栓管理系统是一个基于Web的综合管理系统，旨在帮助学校有效管理和监控校园内的消防栓设施。系统提供了消防栓地理信息可视化、实时状态监控、数据分析统计以及用户管理等功能。

## 系统功能模块

### 1. 用户认证模块

#### 1.1 用户登录
- 访问系统地址，进入登录页面
- 输入用户名和密码
- 点击"登录"按钮进入系统主界面

#### 1.2 用户注册
- 在登录页面点击"点击注册"链接
- 填写用户名（5-10位字符）
- 设置密码（6-15位非空字符）
- 确认密码
- 点击"注册"按钮完成注册

### 2. 仪表盘模块

#### 2.1 三维地图展示
- 系统首页展示基于Cesium的三维校园地图
- 地图包含校园边界、道路、建筑物和消防栓等要素
- 使用Mapbox暗色主题作为底图

#### 2.2 消防栓状态可视化
- 不同颜色标识消防栓状态：
  - 绿色：正常运行
  - 红色：故障状态
  - 橙色：维修中
- 点击消防栓可查看详细信息

#### 2.3 实时数据监控
- 系统右侧面板显示各类统计数据：
  - 正常消防栓数量
  - 压力异常消防栓数量
  - 维修中消防栓数量
  - 实时时间显示

#### 2.4 故障设备列表
- 展示故障和维修中消防栓的详细列表
- 可直接点击查看设备在地图上的位置

#### 2.5 数据分析图表
- 设备状态数量饼图：显示各种状态下消防栓的数量分布
- 压力分布柱状图：显示不同压力区间的消防栓数量
- 平均压力变化折线图：显示压力随时间的变化趋势

### 3. 地理数据管理模块

#### 3.1 消防栓点数据管理
- 在左侧导航栏选择"地理数据管理" -> "消防栓点数据"
- 地图视图展示所有消防栓位置
- 支持多种底图切换（矢量图/影像图）

##### 3.1.1 查看消防栓信息
- 点击地图上的消防栓图标
- 弹出窗口显示消防栓详细信息包括：
  - 编号
  - 状态
  - 压力值
  - 管理单位

##### 3.1.2 编辑模式
- 点击"编辑模式"按钮进入编辑状态
- 支持三种编辑操作：
  - 添加要素：在地图上点击添加新消防栓
  - 更新要素：点击现有消防栓更新其信息
  - 删除要素：点击消防栓删除该要素

##### 3.1.3 数据保存
- 编辑完成后点击相应保存按钮提交更改
- 系统会提示操作结果

#### 3.2 建筑物面数据管理
- 在左侧导航栏选择"地理数据管理" -> "建筑物面数据"
- 查看校园建筑物的空间分布

### 4. 数据表管理模块

#### 4.1 用户数据表管理
- 仅管理员账户可访问此功能
- 在左侧导航栏选择"数据表管理" -> "用户数据表"
- 显示所有系统用户的列表信息

##### 4.1.1 用户信息查看
- 表格展示用户基本信息：
  - ID
  - 用户名
  - 权限等级
  - 性别
  - 出生日期
  - 部门
  - 电话
  - 邮箱
  - 创建时间

##### 4.1.2 用户信息编辑
- 点击用户行的"编辑"按钮
- 在弹出对话框中修改用户信息
- 点击"提交"保存更改

##### 4.1.3 用户删除
- 点击用户行的"删除"按钮
- 确认删除操作
- 系统将移除该用户信息

##### 4.1.4 分页浏览
- 支持分页查看大量用户数据
- 可调整每页显示条目数量

### 5. 用户个人信息模块

- 点击顶部导航栏的用户头像
- 选择"个人中心"查看个人信息
- 可在此处修改个人资料和退出登录

## 系统操作指南

### 基本操作流程

1. **登录系统**
   - 打开浏览器访问系统网址
   - 输入用户名和密码登录

2. **浏览消防栓状态**
   - 登录后自动进入仪表盘页面
   - 查看地图上的消防栓分布和状态
   - 浏览右侧面板的统计数据和图表

3. **查看详情信息**
   - 点击地图上的消防栓查看详细信息
   - 或在右侧面板的故障列表中点击查看

4. **管理消防栓数据**（需相应权限）
   - 进入控制台->地理数据管理->消防栓点数据
   - 开启编辑模式进行数据维护

5. **管理用户信息**（仅管理员）
   - 进入控制台->数据表管理->用户数据表
   - 进行用户信息的增删改查操作

### 注意事项

1. 系统会自动保存登录状态，长时间未操作可能需要重新登录
2. 编辑模式下进行的操作需要手动保存才会生效
3. 删除操作不可逆，请谨慎操作
4. 图表数据会定期更新，反映最新的消防栓状态
